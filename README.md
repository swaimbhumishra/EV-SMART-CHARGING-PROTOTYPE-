Sustainable EV Smart Charging Prototype (PV + WPT + IoT)

A prototype EV micro-charging system integrating solar PV, inductive wireless power transfer (WPT), and an ESP32-based IoT monitoring layer.
This project demonstrates a compact, safe, and intelligent charging architecture designed for decentralized EV charging applications.
